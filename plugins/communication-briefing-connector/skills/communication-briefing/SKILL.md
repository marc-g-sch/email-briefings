# Communication Briefing

Use the local CLI or MCP tools in this plugin to check connector status, ingest supported exports, and generate morning or evening digest previews.

Prefer legitimate data paths:

- Gmail through Codex Gmail tools.
- LinkedIn through approved APIs, partner platforms, or user data export files.
- WhatsApp through Business Cloud webhook payloads or exported chat files.

Do not claim that LinkedIn or WhatsApp personal inboxes were checked unless their connector status is `connected`.
